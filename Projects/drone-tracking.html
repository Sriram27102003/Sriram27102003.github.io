Autonomous Drone Tracking using Deep Learning

Problem Statement

Tracking multiple objects in real-time using a drone is challenging due to fast movement, occlusions, and varying perspectives. This project focuses on developing an intelligent system for robust drone-based object detection and tracking.

Technical Approach

  • Integrated YOLOv3 (You Only Look Once v3) for efficient, real-time object detection.
  • Used Deep SORT (Simple Online and Realtime Tracking with a Deep Association Metric) for multi-object tracking.
  • Data augmentation and preprocessing for real-world aerial video inputs.
  • Tested solutions on simulated and real drone footage streams.

Technologies Used

Python, OpenCV, TensorFlow, YOLOv3, Deep SORT

Results

Achieved robust multi-target tracking in diverse conditions, handling occlusions and high-speed movement. The system successfully maintained real-time performance (<40ms per frame) on GPU-accelerated hardware.
